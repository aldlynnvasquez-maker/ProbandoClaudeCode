# Logos de Partidos Políticos

## Ubicación
Coloca aquí los logos de los partidos políticos en formato PNG.

## Nombres de archivos requeridos:

Los nombres DEBEN coincidir exactamente con los que están en la base de datos:

1. `partido_democratico.png` - Partido Democrático Nacional
2. `alianza_popular.png` - Alianza Popular
3. `verde_progresista.png` - Movimiento Verde Progresista
4. `frente_unido.png` - Frente Unido por el Perú
5. `partido_libertad.png` - Partido Libertad y Democracia
6. `accion_nacional.png` - Acción Nacional
7. `renovacion_popular.png` - Renovación Popular
8. `peru_libre.png` - Perú Libre
9. `fuerza_popular.png` - Fuerza Popular
10. `juntos_peru.png` - Juntos por el Perú

## Especificaciones técnicas:

- **Formato**: PNG con fondo transparente (recomendado)
- **Tamaño**: 100x100px a 200x200px
- **Peso**: Menos de 50KB por archivo
- **Calidad**: Alta resolución para visualización en pantalla

## Ejemplo de logo temporal

Si no tienes logos, puedes crear imágenes temporales con colores de fondo diferentes para cada partido.

## Actualizar logos después de agregar archivos:

Una vez que coloques los archivos PNG en esta carpeta, simplemente reinicia el servidor Flask:

```bash
python app.py
```

Los logos se cargarán automáticamente desde la ruta configurada en la base de datos.
